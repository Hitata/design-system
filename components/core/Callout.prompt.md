**Callout** — an inline note/aside; the heavy left rule is the one place a colored edge earns its keep because it carries meaning.

```jsx
<Callout tone="primary" title="Dụng thần">
  Nourish the scarce elements — Kim and Thủy.
</Callout>
<Callout tone="success" title="Locked">Tokens ported to source of truth.</Callout>
```

Tones: `primary` / `success` / `ink` / `premium` / `neutral`. Flat plaster surface, hairline frame, 4px tone-colored left edge.
